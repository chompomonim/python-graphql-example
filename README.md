Python GraphQL server example
=============================

App was written as example for my talk at [VilniusPy](https://www.meetup.com/vilniuspy/events/239818026/).

Used packages:

* Sanic
* Graphene
* Sanic-GraphQL


Installation
------------

  # docker-compose build
  # docker-compose up

Now in browser open `http://localhost:4000`
